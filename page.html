Facial Recognition Payment System

A self-checkout style payment system where a user’s face is scanned, authenticated against a central database, and a payment is processed automatically without cards or PINs.

Project Overview

This system is designed for environments such as supermarkets, campuses, transport hubs, or secure facilities where fast, frictionless transactions are required.

Key States

  • Scanning: Camera detects and tracks the user’s face
  • Authentication: Facial features matched against database
  • Payment Processing: Linked wallet or account charged
  • Success / Failure: Transaction result displayed clearly

Core Components

  • Camera + Face Detection Model
  • Central Authentication Database
  • Payment Gateway Integration
  • Real-time UI Feedback System
